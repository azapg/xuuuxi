# Agent Instructions for Xuuuxi

## Project Context
- **Name**: Xuuuxi
- **Stack**: React (Vite) for frontend, Bun + SQLite for backend.
- **Goal**: A premium multiplayer card game experience (Cards Against Humanity style).
- **Styling**: Vanilla CSS with modern aesthetics (dark mode, glassmorphism, accent colors). We strictly use `hugeicons-react` for icons. Do **not** use emojis in the UI.

## Workflow Rules
- **Committing Changes**: If given permission, it's highly encouraged and perfectly okay to make frequent commits for all changes. This helps keep track of progress, ensures we don't lose work, and maintains a clean history. You do not need to ask for permission to commit if it falls within the current task scope.
- **Verification**: Always verify your work (e.g., check that the Vite and Bun dev servers are compiling without errors, confirm visual changes if possible) before declaring a task complete.
- **UI Work**: ALWAYS reference the `DESIGN.md` file before starting any UI work to ensure you follow the design philosophy, iconography, and component guidelines.
